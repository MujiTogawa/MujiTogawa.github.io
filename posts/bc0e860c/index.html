0x01 逢魔时

Pt. I

陈春成在他的小说集《夜晚的潜水艇》（《竹峰寺》）中有这么一段文字，描绘得很生动

黄昏时我总爱在寺门外的石阶上坐着，看天一点一点黑下来。想到“苍然暮色，自远而至，至无所见而犹不欲归。心凝形释，与万化冥合”，这些字句像多年前埋下的伏笔，从初中课本上，或唐代的永州，一直等到此时此地，突然涌现。山下的村庄，在天黑前后，异常安静。直到天黑透，路灯亮了，才又听见小孩的嘶喊声。本培说，这村里有个说法，说是人不能在外面看着天慢慢变黑，否则小孩不会念书，大人没心思干活。我记起小时候似乎也听奶奶说过类似的话。山区里，古时山路阻隔，往往两村之间，口音风俗都有所差异，但毕竟同在一县，相似处还是较多。为什么会有这种说法呢？天黑透了却不忌讳，小孩一样玩耍，大人出来乘凉。忌讳的是由黄昏转入黑夜的那一小会。也许那时辰阴阳未定，野外有什么鬼魅出没？我想象在黄昏和黑夜的边界，有一条极窄的缝隙，另一个世界的阴风从那里刮过来。坐了几个黄昏，我似乎有点明白了。有一种消沉的力量，一种广大的消沉，在黄昏时来。在那个时刻，事物的意义在飘散。在一点一点黑下来的天空中，什么都显得无关紧要。你先是有点慌，然后释然，然后你就不存在了。那种感受，没有亲身体验，实在难于形容。如果你在山野中，在暮色四合时凝望过一棵树，足够长久地凝望一棵树，直到你和它一并消融在黑暗中，成为夜的一部分——这种体验，经过多次，你就会无可挽回地成为一个古怪的人。对什么都心不在焉，游离于现实之外。本地有个说法，叫心野掉了。心野掉了就念不进书，就没心思干活，就只适合日复一日地坐在野地里发呆，在黄昏和夜晚的缝隙中一次又一次地消融。你就很难再回到真实的人世间，捡起上进心，努力去做一个世俗的成功者了。因为你已经知道了，在山野中，在天一点一点黑下来的时刻，一切都无关紧要。知道了就没法再不知道。

黄昏与黑夜的交界之时，被日本阴阳道称为鬼神最容易出没的时候，也是人与鬼怪可以同时出现的时刻。所以就有了“逢魔时”的名字（逢魔ヶ時）。
今年我开始走读，开始了每天放学（16:40）便冲进食堂，草率完事后再冲刺到校门口的生活。只为赶上17:05左右到达的那一半班 壹捌玖 路公交车。只要不下雨，傍晚的残阳总会照在路边绿化树光秃秃的枝丫上。秋分过后，天色暗下的速度越来越快。随着太阳直射点的缓慢南移，黑夜，从白天开始变冷。绿化树 枝丫的黑影，逐渐地被拉长，拉长。如同从克苏鲁神话中走出的不明生物一样，向道路上往来的行人车辆探出细长的触手。早年听长辈说起过，迷信说法中，将病人喝剩的药渣倒在闹市路上让行人踩踏，病人便可以延续生命。这我自是不相信的，只是觉得乱丢垃圾很没素质。但在此时此刻的天空下，紫红、橙黄的余晖被渐渐蚕食，交织着冰灰的深蓝色从远山的阴影里爬出，匍匐在天空的一隅，伺机而动。街灯亮了，黑夜从我的体内上升。在这样的环境下，哪怕是风吹草动都足以令人战栗，更别谈这些蜿蜒的灰暗阴影了。我小心翼翼地避开阴影，一步步试探着，不愿被它们吸去灵魂。 壹捌玖 在此时匆匆到站，我却还在马路的对岸。急忙穿过斑马线，刚想停下来伸出手臂拦车，便飞驰而过，对我不加理睬。一阵呛人的黑烟消散后，公交车消失在了不知名的黄昏深处。
落寞地走向站台，宣传橱窗的玻璃不知何时碎了，劣质印刷的图片出露在外。对面站台的LED灯坏了，一直闪烁不停。等车的人们，神色冷漠，像是在等死。
今日又是与公交车擦肩而过的一日。现在，可以很清晰地感知到，那根丝线，在步步紧绷。

Pt. II

今年在时间点上有两件大事，一是我进入了高三，并即将结束第一学期；二是我迎来了我的十八岁。
十八岁似乎是晦明的界线，感觉就像在摆渡过河流。“人无法再度迈进同一条河流”，这句话说得不错的。茫茫河水无边，身前、身后都是苍白无垠的世界，此岸的形状逐渐化为一片无法企及的幻影，而彼岸的轮廓却愈渐清晰起来，将我与过去隔绝。每一次选择都伴随着失去，而如今永远地失去了此岸，我再也无法回到从前了。明日不可知，我仍然身陷渡河的困境。在这进退两难之中，我感觉我正在被缓慢地撕裂。与部分陷入恶性竞争的同学不同，我是闲人，“心” 早就 “野掉了” ，完全无法接受被迫的高压生活。年长者总是会说：“那你又能怎么办呢？”这类完全不像样的安慰。




他妈的，要是我知道方法还会来找你们诉苦啊😧，真是浪费时间。

0x03 展望未来

百无聊赖的高中生活无非也只有这些日子，2025我就要迎来不可知的大学生活。
但可以确定的是，我将会拥有大量的可供个人自由安排的时间。
由于我高考选课（政、地、技），无法选择理科类专业，再加上我对考公考编，法学专业不感兴趣。所以，除去届时课内所学，我打算自己拓展一下技能树，毕竟我现在处于几乎啥都不会的阶段。

一体机计划

技术相关

阶段性目标：

造轮子，自己编写一个静态博客生成器。不想再继续折腾魔改Hexo-Next了，就把这个当作未来的一个目标吧。

需要做什么：

  1. 前端开发
  • 学习 HTML/CSS，掌握页面结构和样式，学习 CSS 框架如 Bootstrap 或 Tailwind CSS 来加速开发。
  • 学习 JavaScript 和前端框架（如 React 或 Vue）来处理动态交互。
  1. 后端开发
  • 学习后端语言（Node.js、Go等）来编写后台部分，实现博客生成逻辑。
  • 学习如何解析 Markdown，将文章从 Markdown 或其他格式转换成静态的 HTML 文件。
  1. 其他
  • 学习 SEO 和 性能优化 技术来提升网站质量。
  • 学习运维基础
